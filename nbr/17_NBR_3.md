```python
tripped = False

    # ...
    if score >= quality_threshold:
        print(f"✅ …")
        tripped = True
        break

if not tripped:
    print(f"\n⚠️ Max iterations reached — final draft was not re-scored")
```

**Blocks transfer:** Yes — the for‑else pattern is non‑obvious and a student trying to write their own loop would likely get the termination handling wrong without understanding the semantics.
